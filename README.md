# MovieSearchEngine

- angular CLI version: 16.2.11
- node version 18.16.0

# Run app
- **npm install** and 
  - **ng serve** for dev mode
  - **ng build** for create production build

# Build docker
- need docker and docker compose
- **docker compose up** for build and run docker
